# Student Marks Analysis Mini Project

## About

This is my Python mini project based on student marks analysis. In this project I generated marks for 50 students and used Python to perform different calculations and display the results.

The main purpose of this project is to practice Python concepts like lists, loops, functions and basic statistics.
## Features

- Generate marks for 50 students
- Calculate total marks
- Calculate average marks
- Find highest and lowest marks
- Count students scoring above 75, below 35 and between 35-75
- Assign grades
- Display top 5 marks
- Display bottom 5 marks
- Find median and standard deviation
- Create a 5-subject marks matrix
- Calculate total marks
- Find topper in each subject
- Generate report card with Total, Average, Percentage and Grade
- ## Concepts Used

- Python Lists
- Loops
- Functions
- Conditional Statements
- Random Module
- Basic Statistics
- ## Conclusion

This project helped me understand how Python can be used to analyze student marks and perform different calculations. It also improved my understanding of loops, functions and lists.
